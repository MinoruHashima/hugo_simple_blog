# コーディングサマリー

## 1. システム全体構造

### アーキテクチャ
- システムタイプ：Hugo静的サイト
- フロントエンド：HTML/CSS/JS（Hugo テンプレート）
- バックエンド：なし（静的サイト）
- データベース：なし

### ディレクトリ構成（主要のみ）
```
hugo_simple_blog/
├── content/        # Markdownコンテンツ
├── layouts/        # Hugoテンプレート
├── static/         # 静的アセット
├── themes/         # テーマ
└── public/         # ビルド出力
```

### データフロー
```
Markdown → Hugo Build → 静的HTML → ブラウザ
```
