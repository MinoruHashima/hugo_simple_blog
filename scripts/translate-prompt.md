# 翻訳プロンプトテンプレート

以下のプロンプトを Claude.ai に貼り付けて使用してください。

---

## 英語翻訳用プロンプト

```
以下の Hugo 記事を英語に翻訳してください。

ルール:
1. frontmatter の title と summary を翻訳
2. 本文（---の下）も翻訳
3. date, external_url, source, country は変更しない
4. 自然な英語表現を使う（直訳を避ける）
5. 出力は Markdown 形式で、そのままファイルとして保存できる形式で

---
元の記事:
---

[ここに日本語記事の内容を貼り付け]
```

---

## 引用元言語への翻訳用プロンプト

```
以下の Hugo 記事を[言語名]に翻訳してください。

ルール:
1. frontmatter の title と summary を翻訳
2. 本文（---の下）も翻訳
3. date, external_url, source, country は変更しない
4. 自然な[言語名]表現を使う
5. 出力は Markdown 形式で、そのままファイルとして保存できる形式で

---
元の記事:
---

[ここに日本語記事の内容を貼り付け]
```

---

## 翻訳後のファイル保存

- 英語版: `post-XX.en.md`
- クロアチア語版: `post-XX.hr.md`
- フランス語版: `post-XX.fr.md`
- など

保存先: `content/posts/`
